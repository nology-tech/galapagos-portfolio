2. MORSE ENCODER

A simple web app which converts a string of latin characters into international morse code.

Try out MORSE ENCODER

This was a three-hour tech test to which I have added some styling. There were many approaches to solving this.

I took the opportunity to practice array iterators and functional programming in order to solve this.

The 'translator' is a simple function which:

▪ converts the user's input from a string into an array

▪ loops through this array

▪ finds the index of the current character in the Latin Alphabet array

▪ returns the character from the same index in the Morse Code Alphabet array

▪ adds the character to a string

▪ inserts the string onto the page

Tech:

2. N.A.M.B.D.

Not Another Movie Database!

A React app using the OMDB API
Visit "Not Another Movie Database"

This was an eighteen-hour tech test, and I left it as it was when I submitted it.

Using Fetch, OMDB and React, I built a simple page which displays movie posters and titles in a tasteful and stylish manner.

I used the following techniques to get it working and looking decent:

▪ React classical components to manage state and the component lifecycle

▪ CSS greyscale filter with hover effects to give the site a "classic film"-feel

▪ Reach Router to configure a nice landing page, the main search page, and an additional page to filter the results by year of production

▪ Array "map" and "filter" iterators with conditional rendering for a smooth user experience

▪ Custom embedded fonts to enhance the look and feel
